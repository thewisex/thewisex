<br/>

Computational pipeline for virtual gene cloning using Python, Biopython, and SnapGene integration.

- Predicts restriction enzyme cut sites across gene sequences
- Optimizes codon usage for target expression systems
- Models protein folding using AlphaFold API integration
- Automates vector design and ligation site analysis

**Stack:** `Python` `Biopython` `SnapGene` `AlphaFold` `Jupyter`

</details>

<details>
<summary><b>🤖 Biological Age Predictor</b> &nbsp;·&nbsp; <code>AI · Longevity</code> &nbsp;·&nbsp; 🔬 Active</summary>
<br/>

Machine learning model to estimate biological age from multiple biomarkers, inspired by epigenetic clock research.

- Trains on synthetic multi-biomarker health datasets
- Implements gradient boosting and neural net age estimators
- Compares against Horvath's DNA methylation clock methodology
- Visualizes healthspan trajectory projections

**Stack:** `Python` `scikit-learn` `TensorFlow` `Pandas` `Matplotlib`

</details>

<details>
<summary><b>🧠 Consciousness in AI — Theoretical Framework</b> &nbsp;·&nbsp; <code>Philosophy · AI</code> &nbsp;·&nbsp; 📝 Research</summary>
<br/>

Theoretical framework evaluating whether artificial systems could possess phenomenal consciousness, drawing on IIT and IGNOU Philosophy of Mind coursework.

- Formalizes Tononi's Φ (phi) metric for neural network topologies
- Applies Chalmers' hard problem criteria to transformer architectures
- Proposes testable predictions derived from Global Workspace Theory
- Bridges analytic philosophy and computational neuroscience

**Stack:** `Python` `NetworkX` `LaTeX` `Jupyter`

</details>

<details>
<summary><b>⚡ EEG Attention Classifier</b> &nbsp;·&nbsp; <code>Neuroscience</code> &nbsp;·&nbsp; 🔬 Active</summary>
<br/>

Exploring EEG signal processing and classification of cognitive attention states using publicly available neurological datasets.

- Implements bandpower feature extraction (alpha, beta, theta waves)
- Trains SVM and LSTM classifiers on attention/distraction states
- Visualizes real-time signal topographies
- Benchmarks against existing BCI literature

**Stack:** `Python` `MNE` `scikit-learn` `NumPy` `Matplotlib`
